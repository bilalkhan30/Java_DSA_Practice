## 📂 Project Structure

The project follows a standard production-ready layout to separate source code from compiled binaries and external libraries:

```text
📦 java_dsa_practice
 ┣ 📂 bin              # Compiled Java bytecode (.class files) [Ignored by Git]
 ┣ 📂 lib              # External third-party dependencies (.jar files)
 ┣ 📂 src              # Source code (.java files)
 ┃ ┗ 📜 MainApp.java    # Application entry point
 ┗ 📜 .gitignore       # Prevents binary and configuration clutter
```

---

## 🛠️ Setup & Execution

### Prerequisites
Make sure you have the Java Development Kit (JDK 11 or higher) installed on your system.

### Compiling and Running via Terminal

1. **Navigate to the root directory:**
   ```bash
   cd /path/to/java_dsa_practice
   ```

2. **Compile the source files into the `bin` directory:**
   ```bash
   javac -d bin src/MainApp.java src/*/*.java
   ```

3. **Run the application entry point:**
   ```bash
   java -cp bin MainApp
   ```
